Descrição Geral do Sistema:
Nosso sistema é um gerenciador de campeonato do jogo Valorant, desenvolvido em JAVA. Ele permite o usuário cadastrar equipes e seus jogadores,
registrar partidas entre equipes, e acompanhar estatísticas de desempenho (KILLS, MORTES, ASSISTÊNCIA & KDA) de cada jogador ao longo do campeonato/partida.
